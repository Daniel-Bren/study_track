# StudyTrack

Aplicação web de gestão de estudos desenvolvida em Python com Streamlit e Supabase.

## Acesse o app

[Clique aqui para acessar o StudyTrack](https://thestudytrack.streamlit.app)

## Funcionalidades

- Autenticação — cadastro e login com email e senha
- Grade de Estudos — monte seu cronograma semanal por edital
- Edital Verticalizado — acompanhe tópicos e subtópicos com checklist de progresso
- Sessão de Estudo — cronômetro e temporizador integrados com as disciplinas do dia
- Progresso — acompanhe seu avanço por disciplina diretamente na Home
- Edital ativo — defina qual edital está estudando e sincronize todas as telas

## Tecnologias

- Python 3.11
- Streamlit
- Supabase (PostgreSQL + Auth)
- Git + GitHub
